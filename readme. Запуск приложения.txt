Приложение по прогнозированию прочности при растяжении материала
1) Запустить VS Code WSL Ubuntu.
Python 3.12.12
conda 25.11.1
2) Запустить окружение conda activate n-0702206
3) Запустить в терминале app.py это сервер приложения на Flask. Далее внести параметры в веб-представлении и нажать "Получить решение". Сервер на Flask должен быть постоянно запущен.
Путь: \\wsl.localhost\Ubuntu\home\pl-an\composite-material-predictor\flaskAPP.
Приложение может принимать на вход параметры из веб-представления, от сторонней системы по JSON API, из десктопного приложения по JSON API.
4) Для запуска имитации отправки запроса по API от сторонней системы нужно запустить в отдельном терминале simpleAPP.py. При этом сервер на Flask должен быть постоянно запущен.
Путь: \\wsl.localhost\Ubuntu\home\pl-an\composite-material-predictor\pythonAPP
5) Для запуска десктопного приложения, в отдельном терминале нужно запустить desktopAPP1.py. Внести параметры, затем нажать "ОК". При этом сервер на Flaskе должен быть постоянно запущен.
Путь: \\wsl.localhost\Ubuntu\home\pl-an\composite-material-predictor\DesktopAPP
